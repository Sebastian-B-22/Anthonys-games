### LORE 📖

**GLiTCH Studios** - A video game company that made games starring GLiTCH and friends.

They wanted their games to be super interactive, so they created an AI called **COMPASS** to give the characters real feelings. But the code had a glitch that corrupted the characters into their **GLiTCHED** versions — twisted, monstrous forms.

**The tragedy:** Some characters are still CONSCIOUS inside their corrupted forms:
- **MAXX** says "...HELP ME..." and "bro... BROTHER... IS THAT YOU..."
- **FLUFFY** is also screaming to get out

**Family connection:** MAXX and CHEFF are BROTHERS. Imagine CHEFF having to face his corrupted brother... 😭

**Gameplay:** Players go through levels, **hacking computers** while avoiding GLiTCHED characters, trying to **fix COMPASS** and save everyone.

**The goal:** Fix the AI (COMPASS) to uncorrupt all the characters and free them.

### Game Concept
- Each character has a NORMAL version (playable) and a GLiTCHED version (evil/enemy)
- Survival horror gameplay inspired by Dandy's World
- The contrast between cute normal versions and terrifying GLiTCHED versions is the core horror element
- Anthony has drawn detailed character designs for all of them

### Stats System
| Stat | What it does |
|------|-------------|
| **Speed** | Movement speed |
| **Hack Speed** | How fast the hack completes |
| **Health** | How many hits you survive (HP) |
| **Script Skip** | Puzzle target size + completion bonus |

### Character Stats (known so far)
**MAXX** 🚒
- Speed: 5 | Hack Speed: 3 | Health: 2HP | Script Skip: 1
- Passive: Fires don't spawn while he's alive

**GLiTCH** 📺
- Speed: 5 | Hack Speed: 7 | Health: 3HP | Script Skip: 7
- Ability: Create temporary HP
- HARDEST TO UNLOCK: 100% data on GLiTCHED GLiTCH + 3333 USBs + 75% data on all main characters

### Currency & Progression
- **USBs** 💾 - Main currency
- **Data** 📊 - Earned by encountering GLiTCHED characters
  - Unlocks: character bios (lore) + power-ups (like Hollow Knight charms)
  - Need both USBs + Data % to unlock characters

### Gameplay Loop
1. Go through levels hacking computers
2. Skill check puzzles pop up (timing-based, circle mechanic like Dandy's World)
   - Solve = faster completion
   - Fail = GLiTCHED enemies alerted to your location
3. Avoid GLiTCHED characters while hacking
4. Complete all machines to progress
5. Ultimate goal: Fix COMPASS to save everyone

### Reference
- Dandy's World is the main inspiration (Anthony sent gameplay videos)
- Skill check timing mechanic directly inspired by Dandy's World extraction puzzles
